## Offensive Security Intro

Learned the purpose of Offensive Security and how ethical hackers identify and exploit vulnerabilities to help organizations improve their security.

---

## Defensive Security Intro

Learned the basics of Defensive Security, including monitoring systems, detecting threats, responding to incidents, and containing attacks to prevent further damage.

---

## Careers in Cyber

Learned about the main cybersecurity career paths and the different roles involved in protecting, testing, and managing computer systems.

---

## Inside a Computer

Learned the basic components of a computer, how they work together, and the boot process from powering on the computer to loading the operating system.

---

## Computer Types

Learned about the different types of computers and their purposes, including desktops, laptops, servers, mobile devices, embedded systems, and mainframes. Also learned that each type is designed for specific tasks based on its performance, size, portability, and intended use.

---

## Client-Server Model

Learned how the client-server model works, where clients send requests and servers process them and return responses. Also learned that most internet services, such as websites, email, and file sharing, rely on this architecture to provide resources and services to multiple users.

---

## Virtualisation Basics

Learned what virtualization is and how it allows multiple virtual machines (VMs) to run on a single physical computer using a hypervisor. Each VM has its own operating system and resources while sharing the same hardware.

Also learned the difference between Type 1 and Type 2 hypervisors, the advantages of virtualization such as isolation, efficient resource usage, snapshots, and easier testing, and why virtual machines are widely used in cybersecurity to safely analyze malware, practice attacks, and build lab environments.
