# Websites

## Overview

A website is a collection of interconnected web pages that can be accessed through a web browser. Websites consist of both **front-end** components, which users interact with, and **back-end** components, which process requests, manage application logic, and communicate with databases.

## Core Technologies

Most websites are built using three core technologies. **HTML** defines the structure of a webpage, **CSS** controls its appearance and layout, and **JavaScript** adds interactivity by responding to user actions and modifying page content within the browser.

## Sensitive Data Exposure

Websites should never expose confidential information to clients. Sensitive data such as credentials, API keys, configuration files, or internal information may be accessible through page source, comments, or improperly configured resources, creating opportunities for attackers during reconnaissance.
