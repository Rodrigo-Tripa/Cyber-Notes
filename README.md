Personal cybersecurity knowledge base containing structured notes, learning summaries, and reference material collected throughout my cybersecurity journey.

Although many notes originate from platforms such as TryHackMe, this repository is designed to be useful for anyone learning cybersecurity, regardless of the learning platform they use.

---

## Overview

This repository serves as a centralized collection of cybersecurity notes covering both fundamental and advanced topics.

The objective is to build a long-term knowledge base rather than simply store course summaries. Concepts are continuously reviewed, expanded, and refined as new knowledge is acquired.

Learning progress from platforms such as TryHackMe is documented separately, allowing the conceptual notes to remain independent from any specific course.

## Note Organization

The repository separates permanent knowledge from learning progress.

- **Concept Notes** explain technologies, protocols, security principles, and methodologies.
- **Learning Notes** summarize what was learned from specific learning platforms.
- **Cheat Sheets** provide quick references for commands and common tasks.
- **Walkthroughs** document practical exercises and machines.

This organization keeps the knowledge base clean while allowing continuous improvement as new concepts are learned.
