# Overview

A **Local Area Network (LAN)** connects devices within a limited geographical area, such as a home, office, or school. LANs provide fast communication and allow devices to share resources efficiently.

## Network Technologies

- Ethernet
- Wi-Fi
- Network Interface Card (NIC)

## Network Devices

- Switch
- Router

## Addressing

- **MAC Address** - Identifies devices within the local network.
- **IP Address** - Identifies devices across different networks.

## Goal

A LAN allows nearby devices to communicate with each other while providing access to external networks through a router.
