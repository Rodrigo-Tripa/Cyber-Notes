# Overview

The **OSI (Open Systems Interconnection) Model** divides network communication into seven layers. Each layer has a specific responsibility, making network protocols easier to understand, implement, and troubleshoot.

## OSI Layers

| Layer | Name |
|------:|----------------|
| 7 | Application |
| 6 | Presentation |
| 5 | Session |
| 4 | Transport |
| 3 | Network |
| 2 | Data Link |
| 1 | Physical |

## Encapsulation

As data moves through the OSI Model, each layer adds its own header in a process called **encapsulation**. The receiving device removes these headers through **decapsulation** to recover the original data.

## Goal

The OSI Model provides a standardized framework for understanding how devices communicate and where different protocols and networking technologies operate.```
