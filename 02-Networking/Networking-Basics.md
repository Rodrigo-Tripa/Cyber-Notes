# Overview

Computer networking allows devices to communicate and exchange data over wired or wireless connections. Networks can range from small local networks to the global Internet.

## Internet vs World Wide Web

- **Internet** - Global network that connects millions of devices.
- **World Wide Web (WWW)** - Service that provides access to websites over the Internet.

## Common Components

- Clients
- Servers
- Switches
- Routers

## Common Protocols

- TCP/IP
- HTTP / HTTPS
- DNS
- ICMP

## Goal

Networking enables reliable communication between devices, allowing users to access websites, transfer files, send emails, and use online services.
