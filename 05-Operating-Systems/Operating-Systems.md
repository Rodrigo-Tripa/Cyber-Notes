## Overview

An Operating System (OS) is software that manages a computer's hardware and provides an interface between users, applications, and the hardware. It is responsible for running programs, managing memory, controlling devices, organizing files, and allocating system resources.

## Main Responsibilities

- Process Management
- Memory Management
- File Management
- Device Management
- User Management
- Security

## Kernel

The kernel is the core component of an operating system. It manages communication between software and hardware, controls hardware resources, and executes system-level operations.

## User Interface

Operating systems provide two main interfaces:

- Graphical User Interface (GUI)
- Command Line Interface (CLI)

## Common Operating Systems

### Linux

Open-source operating system widely used in servers, cybersecurity, cloud computing, and software development.

### Windows

The most widely used desktop operating system, commonly found in personal computers and enterprise environments.

### macOS

Apple's operating system designed for Mac computers.

## Boot Process

The operating system starts after the bootloader loads the kernel into memory. The kernel initializes hardware drivers, system services, and prepares the system for user interaction.
