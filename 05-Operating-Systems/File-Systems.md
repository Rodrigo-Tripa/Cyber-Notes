## Overview

A file system defines how an operating system stores, organizes, and retrieves data on storage devices. It manages files, directories, metadata, and available storage space.

## Common File Systems

### NTFS

The default file system used by Windows, supporting permissions, encryption, compression, and large storage volumes.

### ext4

The most common file system used by Linux distributions, designed for performance, reliability, and journaling.

## Directories

File systems organize data into hierarchical directories, allowing users to efficiently locate and manage files.

## Metadata

Each file stores metadata such as its name, size, permissions, ownership, and timestamps.
