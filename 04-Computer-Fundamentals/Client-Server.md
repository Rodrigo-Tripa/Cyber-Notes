## Overview

The client-server model is a network architecture where clients request services and servers provide them.

## Client

A client is a device or application that sends requests for resources or services.

Examples:

- Web browser
- Email client
- SSH client

## Server

A server processes client requests and provides the requested service.

Examples:

- Web server
- Database server
- File server
- DNS server

## Communication

Clients and servers communicate through network protocols such as:

- HTTP / HTTPS
- SSH
- FTP
- DNS

A single server can handle requests from many clients simultaneously.
