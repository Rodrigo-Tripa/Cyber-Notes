## Overview

Cloud computing provides computing resources such as servers, storage, networking, and applications over the internet instead of requiring local infrastructure.

## Service Models

### Infrastructure as a Service (IaaS)

Provides virtual machines, storage, and networking.

### Platform as a Service (PaaS)

Provides a platform for developing and deploying applications.

### Software as a Service (SaaS)

Provides complete applications that users access through the internet.

## Deployment Models

- Public Cloud
- Private Cloud
- Hybrid Cloud

## Advantages

- Scalability
- Flexibility
- High Availability
- Reduced maintenance
- Cost efficiency through pay-as-you-go pricing
