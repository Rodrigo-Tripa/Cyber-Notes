## Overview

Computer architecture describes how a computer's hardware components work together to execute programs. Every program follows a sequence where it is stored on a storage device, loaded into memory, processed by the CPU, and produces an output.

## Main Components

### CPU

The Central Processing Unit (CPU) executes instructions, performs calculations, and coordinates the operation of the entire computer.

### RAM

Random Access Memory (RAM) temporarily stores the programs and data currently being used by the CPU. It is much faster than permanent storage but loses its contents when the computer is powered off.

### Storage

Storage devices such as HDDs and SSDs permanently store the operating system, applications, and user files.

### Motherboard

The motherboard connects every hardware component, allowing communication between the CPU, RAM, storage devices, and peripherals.

## Boot Process

When a computer starts:

1. The system is powered on.
2. BIOS or UEFI initializes the hardware.
3. The bootloader is executed.
4. The operating system kernel is loaded into memory.
5. Drivers and system services are initialized.
6. The operating system becomes ready for use.

## Data Flow

A program is stored on disk, loaded into RAM, executed by the CPU, and the results are sent to an output device or saved back to storage.
